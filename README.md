# PL Construções & Reformas

Landing Page institucional desenvolvida para a **PL Construções & Reformas**, com foco em geração de leads, apresentação de serviços e conversão de visitantes em contatos via WhatsApp.

## 📋 Sobre o Projeto

O objetivo desta aplicação é fornecer uma presença digital moderna para a empresa, destacando seus serviços, portfólio, diferenciais e canais de contato de forma clara e profissional.

A interface foi projetada seguindo princípios de responsividade, experiência do usuário (UX) e otimização para conversão (CRO).

Veja o projeto rodando aqui: [Clique aqui para acessar](https://pl-pinturas-reformas.vercel.app)

## ✨ Funcionalidades

* Navegação por âncoras entre seções
* Menu responsivo para dispositivos móveis
* Menu hambúrguer com JavaScript
* Botão de contato via WhatsApp
* Efeitos de hover e interações visuais
* Layout totalmente responsivo
* Estrutura otimizada para futuras expansões

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome

## 📱 Responsividade

O projeto foi desenvolvido utilizando a abordagem Mobile First, garantindo uma boa experiência em:

* Smartphones
* Tablets
* Notebooks
* Monitores Desktop

## 🎯 Objetivo de Negócio

A landing page foi construída com foco em:

* Captação de novos clientes
* Solicitação de orçamentos
* Fortalecimento da presença digital
* Divulgação dos serviços da empresa

## 🚀 Melhorias Futuras

* Integração direta com WhatsApp Business
* Galeria dinâmica de projetos
* Formulário de orçamento
* SEO otimizado
* Animações e microinterações
* Painel administrativo para gerenciamento de conteúdo
